- Como usar Github con consola

Descargar Git

- Ponemos en la consola el siguiente comando: git config --global user.name "tu usuario"

- Ahora ponemos el correo electr√≥nico asiociado a nuestra cuenta con el siguiente comando: git config --global user.email correo@gmail.com

- Llevar el repositorio de Github a tu escritorio

- Primero nos situamos en el escritorio con el comando cd, o hacemos click derecho en la carpeta escritorio y de damos a abrir terminal.

- Una vez tengamos la terminal abierta vamos a github y copiamos el enlace para clonar el repositorio

- Cuando copiemos el enlace ponemos el siguiente comando: git clone (el enlace)

- Guardar cambios y subirlos a Github (hacerlos en orden y siempre)

- Para guardar TODO, recomendable hacerlo siempre: git add .

- Commit changes y poner un texto sobre los cambios: git commit -m "texto"

- Para subirlo a Github: git push

- Actualizar el repositorio

- Para actualizar el repositorio de tu escritorio para que coga lo que hay en github y lo ponga pondremos el siguiente comando: git pull
