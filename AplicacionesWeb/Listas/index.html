Listas de elementos en HTML

- Listas Desordenadas con "ul" (No tienen números delante)

Con style="list-style:" puedes escoger el tipo de simbolo que quieras que aparezca antes del texto, en este caso es un circulo

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemento 4

- Listas Ordenadas con "ol" (tiene números)

Con "type=" puedes esscoger el tipo de orden que quieres, en este caso es type = "1" y start 33

  1. Valencia
  2. Teruel
  3. Zaragoza
  4. Albacete
