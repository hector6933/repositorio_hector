Tablas en HTML

- Hacer una tabla con "table", filas con "tr", y para hacer una columna dentro de cada fila "td".
- Con el atributo "width=x%" dento de "table" se puede modificar el ancho de las columnas.
- Si en vez de "td" ponemos "th" se centra y se pone en negrita (como un titulo), pa que este mejor.
- (con css se puede modificar el estilo de los bordes de la tabla y demás)
