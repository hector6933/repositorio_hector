Listas

Para crear una lista ordenada usaremos la etiquta "ol" (ordered list)
Y para crear una lista desordenada usaremos la etiqueta "ul" (unordered list)
Cada apartado en la lista se define con "li" Con el atributo "list-style" podemos cambiar el formato de los iconos que aparecen delante de cada lista. Y se pueden meter "ol" y "ul" dentro de cada "li".

Formularios

Para hacer un formulario abriremos la etiqueta "form"
Dentro de la etiqueta form ya podemos empezar a construri nuestro formulario
Hay x principales etiquetas dentro del formulario:

- Input y label:

Para crear un input usamos la etiqueta "input" y dentro de ella le damos un id ej: id="texto1" Para crear un label de ese input creamos la etiqueta "label" y dentro de ella ponemos for="texto1"


- Fieldset:

Para que quede más guapo le podemos poner dentro del formulario la etiqueta "fieldset" y dentro del fieldset cosas.
Dentro del "fieldset" le podemos poner la etiqueta "legend".
En un mismo formulario podemos poner más de un fieldset.
